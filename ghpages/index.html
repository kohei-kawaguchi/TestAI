TestAI - Analysis Reports

This repository implements statistical analysis and dynamic optimization methods, including causal inference methods in R and Markov Decision Process solvers in Python.

  • Markov Decision Process Solver
    Neural network-based value function approximation using PyTorch. Implements value iteration algorithm with Bellman operator, continuous state space, and comparative statics analysis.
  • Difference-in-Differences Analysis (Multi-Period)
    Implementation of Callaway-Sant'Anna estimator for multi-period difference-in-differences with support for covariates and multiple treatment periods.
  • Neyman's Average Treatment Effect
    Estimation of average treatment effect using Neyman's framework for randomized experiments with robust variance estimation.
